# 🎮 Rock Paper Scissors (JavaScript Game)

A simple and interactive **Rock Paper Scissors** web game built using **HTML, CSS, and JavaScript**.
Play against the computer and try to reach **5 points before it does**.

---

## 🚀 Features

* 🎯 Play against a computer opponent
* 📊 Live score tracking (Player vs Computer)
* 🧠 Randomized computer moves
* 🏁 Game ends automatically at 5 points
* 🔒 Buttons disabled after game ends
* 🔄 Reset functionality to restart the game
* 🎨 Clean and responsive UI with hover effects

## ⚙️ How It Works

* User selects **Rock / Paper / Scissors**
* Computer generates a **random choice**
* Game logic determines winner:

  * Rock beats Scissors
  * Paper beats Rock
  * Scissors beats Paper
* Scores are updated after each round
* First to reach **5 points wins the game**
